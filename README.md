# DominAI

Plataforma de **Energy Intelligence (EI)** escrita em Python, pensada para funcionar como um ERP vertical do setor de energia.

A ideia é simples: juntar o poder do BI à centralização de dados que vêm direto de hardware próprio, para monitorar e otimizar a eficiência energética sem complicação.

---

## O que ele faz

**Centralização de dados**
Recebe e integra métricas estruturadas direto do hardware proprietário, sem intermediários.

**ERP verticalizado**
Um sistema de gestão sob medida para os fluxos específicos do setor de energia — não é um ERP genérico adaptado na marra.

**Business Intelligence**
Gera relatórios analíticos e inteligência de dados para apoiar decisões estratégicas.

**Armazenamento em CSV**
Todo o histórico de dados e sensores fica salvo e estruturado localmente.

---

## Como rodar

Você vai precisar do **Python 3.10** instalado.

```bash
# clone o repositório
git clone https://github.com/Projeto-DominAI/dominai-g6.git

# entre na pasta do projeto
cd dominai-g6

# inicie o sistema
python main.py
```

Pronto — o sistema sobe a partir daí.
